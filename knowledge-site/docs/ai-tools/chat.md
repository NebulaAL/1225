# 对话类 AI 工具对比

## 主流工具一览

### ChatGPT（OpenAI）
- **官网**：chat.openai.com
- **适合场景**：日常问答、写作、翻译、代码
- **优势**：生态最成熟，插件多
- **劣势**：中文偶尔不自然

### Claude（Anthropic）
- **官网**：claude.ai
- **适合场景**：长文分析、代码、严谨推理
- **优势**：上下文窗口大，指令遵循好
- **劣势**：国内访问不便

### 文心一言（百度）
- **官网**：yiyan.baidu.com
- **适合场景**：中文写作、国内业务
- **优势**：中文理解好，无需翻墙
- **劣势**：英文和代码能力稍弱

### DeepSeek
- **官网**：deepseek.com
- **适合场景**：推理、代码、数学
- **优势**：性价比极高，开源可部署
- **劣势**：多模态能力有限

## 怎么选？

| 需求 | 推荐 |
|------|------|
| 日常使用 | ChatGPT |
| 写代码 | Claude / DeepSeek |
| 中文写作 | 文心一言 |
| 省钱 | DeepSeek |
| 隐私敏感 | DeepSeek（本地部署） |

::: info 提示
以上信息基于实测体验，各工具更新频繁，建议关注官方动态。
:::
