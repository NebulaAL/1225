# AI 绘图工具

## Midjourney
- **使用方式**：Discord 机器人 / 官网
- **风格**：艺术感强，出图质量高
- **价格**：$10/月起
- **上手技巧**：关键词用英文，加风格词效果更好

## Stable Diffusion
- **使用方式**：本地部署（免费）或在线平台
- **风格**：高度可控，支持 LoRA/ControlNet
- **价格**：本地免费，需要显卡
- **上手技巧**：推荐用 ComfyUI 工作流

## DALL-E 3（OpenAI）
- **使用方式**：ChatGPT Plus 内置
- **风格**：写实、指令理解准确
- **价格**：包含在 ChatGPT Plus 订阅中

## 对比

| | Midjourney | Stable Diffusion | DALL-E 3 |
|--|-----------|-----------------|----------|
| 出图质量 | ⭐⭐⭐⭐⭐ | ⭐⭐⭐⭐ | ⭐⭐⭐⭐ |
| 上手难度 | 简单 | 较难 | 最简单 |
| 可控性 | 中 | 最高 | 中 |
| 免费 | 否 | 是（本地） | 否 |

::: tip
新手推荐从 DALL-E 3 开始（最简单），进阶选 Midjourney，想深度玩选 Stable Diffusion。
:::
