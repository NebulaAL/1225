# 国内使用 Claude 手册

> 从零开始，一步一步搭建你的 Claude 使用环境
>
> 最后更新：2026/3/22

## 一、前置条件

![前置条件 — 设备准备](/images/image_generated_20260420_115526_569_0_5f4fbc2a-8cce-4ba5-93e6-f78294c32541.png)

开始操作前，请确认以下条件均已满足：

- 一台 Mac 电脑（主要操作均在 Mac 端完成）
- 一部 iPhone 或 iPad（通过 App Store 内购订阅 Claude Pro 必需）
- 已搭建或购买可靠的 VPN，并确保节点 IP 纯净

::: warning 注意
Mac 负责注册和日常使用，iPhone/iPad 负责 App Store 订阅支付，两者缺一不可。
:::

## 二、节点测试

![节点测试 — VPN 检测流程](/images/image_generated_20260420_115540_328_0_ed2333a5-946c-4a35-b681-e7fd15924d0d.jpg)

注册前建议先测试节点是否干净，三个步骤如下：

1. 访问 [Google 首页](https://www.google.com)，能正常加载即说明节点可用。查看页面右下角显示的地区，确认已切换到目标国家。
2. 访问 [ip.sb](https://ip.sb) 或 [whatismyipaddress.com](https://whatismyipaddress.com)，查看当前 IP 地址、归属地和运营商，确认已切换到境外 IP。
3. 访问 [claude.ai](https://claude.ai)，能正常打开说明节点干净，可以开始注册。

::: warning 注意
节点是否纯净是注册成功的核心，其他条件相对宽松。如果测试失败，请更换节点再试。
:::

## 三、准备账号

![准备账号 — Google + App Store](/images/image_generated_20260420_115556_852_0_b8e2d5e2-5397-42ef-ae76-41cdaf17167e.png)

### 3.1 Google 账号

注册 Claude 需要一个 Google 账号登录。如果没有美区 Google 账号，可在淘宝购买。

- 账号越老越稳定，不容易被封
- 邮箱内容不重要，节点纯净才是关键

::: danger 重要
- 使用美区 Google 账号和美区 App Store 账号，是为了降低账号被封的风险。
- 使用任何 Google 相关功能时，必须全程开启 VPN，否则容易触发封号机制。
- **重要！一定要用 VPN，保证节点 IP 纯净。**
:::

### 3.2 App Store 美区账号与订阅

Claude 目前在 Mac App Store 可能未上架，Mac 用户可到官网下载桌面版，但官网下载版仅支持信用卡支付。

如需通过 App Store 内购订阅会员，只能在 iPhone 或 iPad 上操作，平台内购需美区 App Store 账号。

| 方式 | 说明 |
|------|------|
| 方式一 | 官网下载 Mac 桌面版，使用信用卡支付订阅 |
| 方式二 | 在 iPhone/iPad 上通过美区 App Store 内购订阅（如没有美区账号可在淘宝购买） |

## 四、注册 Claude 账号

### 4.1 登录 Google 账号

购买 Google 账号后，卖家会提供以下信息：

- 登录邮箱地址
- 登录密码
- 2FA 双重验证信息

#### 2FA 双重验证操作步骤

![注册流程 — 2FA 验证步骤](/images/image_generated_20260420_115613_885_0_a4f16c63-c25b-4613-ad43-a6c2db7d7c18.png)

1. 输入账号和密码后，Google 会提示进行双重验证
2. 打开卖家提供的验证工具网址 [2fa.cn](https://2fa.cn/)
3. 将卖家提供的 2FA Secret 粘贴进去
4. 点击 Submit，获取 6 位数字验证码
5. 将验证码填入 Google 验证框，完成登录

### 4.2 注册 Claude 账号

登录 Google 账号后，前往 [claude.ai](https://claude.ai) 进行注册。

- 使用美区 Google 账号登录
- 过程中会要求进行手机号验证
- 手机验证可在闲鱼购买，美区或英区号码均可，价格约 5.8 元
- 完成验证后即可注册成功

## 五、订阅 Claude Pro

### 5.1 套餐价格一览

| 套餐 | 价格 | 特点 |
|------|------|------|
| 免费版 | $0/月 | 每日消息数量有限 |
| Claude Pro | $20/月 | 额度更大，优先体验新功能 |

> 对于国内普通用户，推荐优先体验免费版，再根据需要评估是否订阅 Pro。

### 5.2 App Store 礼品卡充值

![订阅 Claude Pro — 礼品卡充值](/images/image_generated_20260420_115613_133_0_c12efdd3-8b0e-4359-bed4-80004ef6e9e4.jpg)

国内银联卡无法直接支付美区 App Store，需要先购买礼品卡充值。

- 礼品卡可在淘宝购买
- 充值时会有手续费和汇率损耗，实际到账金额少于购买金额
- 礼品卡充值可能需要等待一段时间，这是平台风控机制导致的正常现象，耐心等待即可

::: warning 注意
充值前建议多准备 10%–15% 的额外预算，避免余额不足无法订阅。
:::

礼品卡充值完成后，在 App Store 中搜索 Claude 并订阅即可。

## 六、常见问题

![常见问题 — FAQ 总结](/images/image_generated_20260420_115636_644_0_8211924c-1d09-431c-ab4a-e76924614b36.png)

### Q: 为什么注册失败？

最常见的原因是 VPN 节点不够纯净。建议更换节点再试，并通过[节点测试步骤](#二、节点测试)确认。

### Q: 手机验证可以用哪些国家的号码？

美区和英区号码均可，闲鱼上一般会标注，价格约 5.8 元。

### Q: 免费版和 Pro 版有什么区别？

免费版每日消息数量有限，Pro 版使用额度更大，并可优先体验新功能。

### Q: 礼品卡充值为什么要等很久？

App Store 礼品卡充值受平台风控影响，充值后可能需要等待一段时间才会到账，这是正常现象。

### Q: 礼品卡余额充足，但仍然无法订阅 Claude？

如果礼品卡里有余额却订阅失败，说明该 Apple ID 已被风控限制。需要拨打苹果客服电话进行人工解封：

::: tip 苹果客服
- **中国大陆**：400-666-8800
- **在线支持**：[getsupport.apple.com](https://getsupport.apple.com)

联系客服时说明"App Store 订阅被限制"，客服会协助解除风控。
:::

### Q: 使用过程中突然被退出登录了？

如果在正常使用时突然被强制退出、且无法重新登录，说明该 Claude 账号已被封号。

::: warning 预防封号建议
- 始终开启 VPN，且保持节点 IP 稳定，避免频繁切换节点
- 不要在短时间内大量发送消息，避免触发异常行为检测
- 尽量使用美区节点，避免使用被标记的高风险 IP
:::

账号一旦被封通常无法申诉恢复，建议重新注册一个账号。
