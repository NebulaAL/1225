# 提示词模板库

> 开箱即用，复制粘贴直接用。

## 通用模板

### 角色扮演
```
你是一个 [角色]，拥有 [年数] 年 [领域] 经验。
请用 [风格] 的方式回答我的问题。
我的问题是：[问题]
```

### 结构化输出
```
请分析 [主题]，按以下格式输出：
1. 背景概述（50字以内）
2. 核心要点（3-5条）
3. 实操建议（具体可执行）
4. 注意事项
```

## 写作类

### 小红书文案
```
请帮我写一篇小红书帖子：
主题：[主题]
目标人群：[人群]
要求：
- 标题带 emoji，吸引点击
- 正文口语化，有互动感
- 结尾带话题标签 3-5 个
- 字数 300-500 字
```

### 公众号文章
```
请写一篇公众号文章：
主题：[主题]
风格：[专业/轻松/故事化]
结构：引子 → 痛点 → 解决方案 → 总结
字数：1500-2000 字
```

## 编程类

### 代码审查
```
请审查以下代码，从这几个角度分析：
1. 是否有 bug 或逻辑错误
2. 性能是否可优化
3. 可读性和命名是否规范
4. 安全性问题

代码如下：
[粘贴代码]
```

### 需求转代码
```
请根据以下需求编写代码：
语言：[语言]
需求：[描述]
要求：
- 代码简洁，有注释
- 包含错误处理
- 提供使用示例
```

## 学习类

### 快速学习
```
我想在 [时间] 内学会 [技能]。
我的基础是：[基础水平]
请给我：
1. 学习路线图
2. 推荐资源（免费优先）
3. 每日练习计划
4. 常见误区提醒
```

::: tip
这些模板是起点，根据实际效果调整 [] 中的内容即可。好的提示词 = 清晰的角色 + 明确的任务 + 具体的格式要求。
:::
