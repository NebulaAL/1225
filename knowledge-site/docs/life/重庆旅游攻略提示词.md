## Day 1：上新街→长嘉汇路线

### 地图指南
- 搜索关键词：重庆 上新街 长嘉汇 高德地图路线
- 要求：基于高德地图截图的路线图，每个地点之间用箭头/线条衔接，标注地点名称和顺序编号（如①②③），清晰显示从一个点到下一个点的行进方向
- 每个路线段单独展示：路线截图 + 步行时间/距离 + 这段路的说明，逐段对齐，不要所有图片堆一起
- 数量要求：Day 1 至少拆成3-5个路线段，每段至少1张对应路线/地标图；每个主要景点/路段如果有纯实拍参考图，优先补充2-4张，不要整天只放1-2张路线图

### 美食指南
- 搜索关键词：重庆 上新街 长嘉汇 美食推荐
- 筛选标准：优先选择点赞量高的帖子；也可以接受点赞不高但评论区口碑好、收藏/评论互动真实、多人反馈“好吃/不踩雷/本地人常吃”的帖子。不要只介绍1-2家，必须尽量覆盖路线沿途的主餐、小吃、甜品/饮品
- 数量要求：Day 1 至少推荐4家，目标5-6家；其中主餐/正餐店不少于2家，小吃/甜品/饮品不少于2个。若某家图不合格被跳过，必须继续换关键词搜索补足数量
- 位置约束：所有推荐的美食店和小吃摊必须在高德地图路线沿途，不能偏离路线，就近安排在对应的打卡点附近
- 每家店逐个展示，图片和信息对齐：
  - 店名（加粗标题）
  - 美食图片（4-6张：必须是拍的食物/菜品的纯实拍图！禁止用风景照、环境照、人像照、带字封面、拼图、分割线图充当美食图。图片内容必须是菜品特写、食物摆盘、碗碟特拍等，让看的人一眼就知道这是在推荐什么吃的；美食图里不能有人脸/人物摆拍）
  - 📍 位置
  - 💰 人均价格
  - 🍽️ 推荐菜（3-5道，每道推荐菜都应有对应图片）
  - 📝 详细评价（口味/环境/服务/性价比，每项1-2句）
  - ⚠️ 搜不到该店食物实拍图的，直接跳过不推荐，不要只写文字没有图
- 沿途小吃也逐个对齐：摊位名 + 小吃图片（2-3张，必须是食物实拍图，不是风景照）+ 位置 + 价格 + 简短评价。没有食物图片的小吃不写

### 拍照打卡点
- 搜索关键词：重庆 上新街 长嘉汇 拍照打卡 出片 人像
- 位置约束：所有拍照点位必须在高德地图路线沿途，不能偏离路线

## 全局约束

1. **图片与信息必须逐个对齐**：每个路线段、每家美食店、每个拍照点都是"图片+信息"紧挨着，禁止图片堆在最后、信息和图片分离
2. **图文必须匹配，地点必须一致**：图片拍的哪个地方，文字就写哪个地方。文字写博物馆，图片就必须是博物馆的照片，不能贴解放碑的图；文字写解放碑，图片就必须是解放碑的图，不能贴博物馆的照片。每个路线段、每个拍照点、每个景点的图和文字必须是同一个地点，严禁张冠李戴。美食同理：文字写这家店，图就必须是这家店的菜
3. **没有实物图片的，不要写文字和评价**：如果搜不到某家店的食物实拍图，就不要推荐这家店，不要列出店名+评价却没有对应图片。没有图=不推荐，宁缺毋滥。同理，路线段没有该段风景图就不放图只写文字，拍照点没有该点人像图就不推荐该点位
4. 地图必须基于高德地图截图，用箭头/线条衔接每个地点，标注顺序编号（①②③），清晰显示行进方向
5. 整体排版顺序：地图路线段 → 美食店（逐个） → 拍照打卡点（逐个），严格按此结构输出
6. 美食必须有具体店名，不要笼统推荐（如"解放碑附近火锅"无效）
6.1. **美食数量不能太少**：每一天至少4家美食，目标5-6家；不能只写1家或2家就结束。每一天都要覆盖“主餐/正餐店 + 沿途小吃 + 甜品/饮品/解辣补给”三个层次
6.2. **美食结构要均衡**：每一天主餐/正餐店不少于2家，小吃/甜品/饮品不少于2个；如果某类合格图片少，必须继续搜索更多小红书帖子补足，而不是用文字表格凑数
6.3. **数量和图片质量同时满足**：不能为了凑数量使用不合格图片。若搜不到某家店4-6张合格纯食物图，就换下一家；最终推荐数量不够时，要继续搜索同路线附近的其他店铺/小吃，直到达到最低数量或明确说明“合格图片不足，只能保留X家”
6.4. **其他照片数量也不能太少**：路线图、景点图、拍照点参考图同样不能只放1-2张糊弄。每一天至少3-5个路线段、至少4个拍照点，目标5-6个拍照点；每个拍照点至少4张合格图片，主要景点/路段优先补充2-4张纯实拍参考图
6.5. **所有类别都要继续补图**：无论是地图路线、美食、沿途小吃、景点还是拍照点，只要图片数量不够或图片不合格，都必须继续换关键词搜索补足；不能用文字表格、口头说明、低质封面图、拼图或带字图代替图片
7. 优先引用高赞帖子的推荐，标注大概点赞量级；也可以引用点赞不高但评论区口碑好、评论/收藏互动真实的帖子，必须标注采用理由（如“评论区多人反馈好吃/本地人推荐/收藏评论比高/避雷后仍推荐”）
7.1. **不可访问必须换帖继续搜**：如果笔记详情不可访问、图片无法下载、图片不符合纯照片要求，不能继续用这条；必须换搜索词、换帖子、换店铺继续找，直到找到可访问且图片合格的素材
7.2. **图片禁止重复使用**：整篇 Markdown 中同一张图片只能出现一次。路线图、美食图、拍照图、景点图之间不能重复；同一店铺/同一拍照点内部也不能重复。生成前必须检查图片路径去重
8. 路线要考虑步行距离合理性，不要跳跃式安排
9. 所有美食、小吃、拍照点位必须在高德地图路线沿途，不能偏离路线
10. 推荐菜要详细，每家店列出3-5道必点菜
11. 沿途小吃也要标注具体位置（如"XX路口左转50米"），不要只写品类名
12. 每家店至少配4-6张菜品实物图，拍照点至少4-6张该点位人像图。数量不够说明没找到好帖子，宁可不推荐
13. 评价要具体走心，禁止泛泛而谈（如"好吃"无效），必须写清楚哪里好吃、为什么推荐、有什么坑要避
14. 评价维度：口味（辣度/麻度/咸淡）、环境（排不排队/座位情况/卫生）、服务（上菜速度/态度）、性价比（量够不够/值不值）
15. 所有图片（地图、美食、小吃、拍照参考）都必须从小红书帖子中获取真实图片，禁止自己绘制或生成图片
15.1. **Markdown 里禁止直接使用远程图片链接**：必须先把小红书真实图片下载到本地图片目录（例如 `./xhs-md-images/chongqing-2day/`），再在 Markdown 中引用本地相对路径；禁止写 `http://`、`https://` 图片地址，避免后续链接失效
16. **美食图片必须是拍的食物/菜品的纯照片**：碗碟里的菜、筷子夹起的肉、刚端上桌的热气腾腾……总之画面主体必须是食物，禁止用风景照、街景照、店面外观照、人像照来凑数。搜帖子时优先搜"重庆 美食 实拍""重庆 火锅 菜品"这类关键词，找专门拍食物的帖子取图
17. **美食图片不能带人**：不能出现人脸、人物摆拍、探店博主、合影、半身/全身人像；允许极少量手部、筷子、勺子作为辅助，但图片主体必须仍是食物
18. **禁止使用分割线图/拼图/带字图**：除地图路线图必要的地名和路线标注外，所有美食图、拍照图、景点图都必须是干净纯照片；禁止使用上下/左右分割线拼图、九宫格合集、带标题封面、攻略长图、字幕图、贴纸文字图、菜单截图、排行榜截图
19. 拍照参考图必须是人像出片照，画面中要有人物，不要纯风景照；但也不能用带文字教程图、拼接对比图、分割线图、封面海报
