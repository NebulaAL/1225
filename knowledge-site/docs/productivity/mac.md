## 快捷键提效

### Raycast 常用
- `Cmd + Space` → 打开 Raycast
- 输入 `calc 100*1.13` → 计算器
- 输入应用名 → 秒开应用
- 剪贴板历史 → 自定义快捷键

### 系统级
- `Cmd + Tab` → 切换应用
- `Cmd + ~` → 同一应用切换窗口
- `Ctrl + 左右箭头` → 切换桌面
- `Cmd + Shift + 5` → 截图/录屏

## iTerm2 分屏
- `Cmd + D` → 垂直分屏
- `Cmd + Shift + D` → 水平分屏
- `Cmd + ]` / `Cmd + [` → 切换面板
